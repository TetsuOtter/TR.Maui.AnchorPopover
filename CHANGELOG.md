# Changelog

## [1.0.0] - 2025-01-18

### Added
- Initial release of TR.Maui.AnchorPopover
- Native popover implementation for iOS/MacCatalyst using UIPopoverPresentationController
- Android implementation using PopupWindow with Material Design styling
- Windows implementation using Flyout controls
- Anchor support for attaching popovers to MAUI views
- Position-based anchoring for displaying popovers at specific screen locations
- Arrow direction configuration (Up, Down, Left, Right, Any)
- Customizable popover options (size, background color, dismissal behavior)
- Comprehensive sample application demonstrating all features
- Full API documentation
- GitHub Actions workflows for CI/CD and NuGet publishing

### Platform Support
- iOS 15.0+
- macOS 15.0+ (via MacCatalyst)
- Android API 21+
- Windows 10.0.17763.0+
